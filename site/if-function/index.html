ifを式として扱う

Kotlinでのifは単なる条件分岐としての役割だけでなく、
式として評価するための構文です。

val hoge = "hoge"
val result = if(hoge == "hoge") { "期待通り" } else { "hogeじゃない"}
println(result) // => 期待通り

Stringの戻り値を持つ、処理のように扱うことが出来ます。
（もちろん、IntでもBigDecimalでも問題ありません）

このように書くことも出来ます

val hoge = "hoge"
val result = 
  if(hoge == "hoge")  "期待通り" 
  else  "hogeじゃない"
println(result) // => 期待通り

条件分岐を行うで書いた処理も
厳密にはUnitという戻り値を持たない処理ための型を返しています。
